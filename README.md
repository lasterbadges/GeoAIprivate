# GeoAIprivate
1. Установка докер контейнера docker pull ghcr.io/lasterbadges/geoaiprivate:latest
2. запуск docker run -d -p 8501:8501 --name geoai ghcr.io/lasterbadges/geoaiprivate:latest
3. откыть браузер по адрессу http://localhost:8501/
4. Загрузить фото в поле микрофотографии шлифа по средством перетягивания офто или нажатия и выбора через контекстное окно
5. Нажать на кнопку начать анализ
6. открыть вкладку Аналитический дэшборд дляс просмотра более подробной информации
